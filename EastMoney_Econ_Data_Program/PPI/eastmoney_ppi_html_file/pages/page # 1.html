工业品出厂价格指数是反映全部工业产品出厂价格总水平的变动趋势和程度的相对数。
重要性
非常高：工业品出厂价格指数是衡量工业企业产品出厂价格变动趋势和变动程度的指数，是反映某一时期生产领域价格变动情况的重要经济指标，也是制定有关经济政策和国民经济核算的重要依据。
